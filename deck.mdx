There are 2 prevailing methods to enhance knowledge understanding, aquisition, and retention.

1. [Spaced repetition](https://pdfs.semanticscholar.org/52d7/5ac6080842077877219003dbfbcbea36020f.pdf)
   - Spaced practice enhances memory, problem solving,
and transfer of learning to new contexts.
2. [Learning by teaching](https://www.sciencedirect.com/science/article/abs/pii/S0361476X13000209?via%3Dihub#ab005)
   - Students who actually taught experienced more persistent learning gains.

Output: 6

### Median of Two Sorted Arrays

> Find the median of the two sorted arrays. 

```
nums1 = [1, 2] nums2 = [3, 4]

The median is (2 + 3)/2 = 2.5

-----------------------------

nums1 = [1, 3] nums2 = [2, 4]

The median is (2 + 3)/2 = 2.5
```

### Linked List Deep Copy (Copy List with Random Pointer)

> A linked list is given such that each node contains an additional random pointer which could point to any node in the list or null. Return a deep copy of the list.

```
Input/output: 
    {
        "$id":"1",
        "next":{
            "$id":"2",
            "next":null,
            "random":{
                "$ref":"2"
            },
            "val":2
        },
        "random":{"$ref":"2"},
        "val":1
    }
```
